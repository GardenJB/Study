## Git 설치 및 환경설정

- git clonfig
    - —local
    - —global
    - —system
    
    ```jsx
    * 작업자 이름 설정
    $ git config --global user.name "username"
    * 작업자 전자메일 설정
    $ git config --global user.email "usermail@hostname"
    
    * 글로벌 설정값 확인
    $ git config --list
    * 로컬 설정값 확인(저장소별 설정)
    $ git config --get user.name
    $ git config --list
    
    * 도움말
    $ git help
    ```
